Welcome to the Forgotten Realms - The Shadowdale Campaign!

This is a campaign set in Shadowdale in the Forgotten Realms campaign setting. Since the game will be using the AD&D 2nd Edition ruleset, the campaign will start off in 1368 DR. This is the setting as presented for use with the AD&D 2nd Edition rules.

To find more information on either Forgotten Realms or Shadowdale, please check the Forgotten Realms Campaign Setting, Revised. For more player-centric information on Shadowdale, refer to the Shadowdale page for a map and items of interest within Shadowdale.

Just a small word of warning: If you use the Internet as a source of information, be aware that you will usually find information on the Forgotten Realms for later years, i.e., the default settings for later editions of D&D. So you need to make sure that whatever you check is valid for 1368 DR.

And finally, be aware that this is my Forgotten Realms, so anything you learn from an "official" source may or may not be true.

Player Introduction

To create your character, please look at the House Rules - Character Creation rules, but otherwise follow all the rules as given in the Player's Handbook.

A note on sources

We will be using the revised version of the rules, the so-called 2.5 version that was reprinted in 2013. You can acquire the Player's Handbook on Drivethrurpg. We will be using the rules as written, except as noted in the House Rules section (see menu above).

One aspect of AD&D 2E is that there is a lot of supplamental material. Although I am not encouraging anyone to pull from these sources, neither am I explicitly disallowing them. If you wish to use anything from a supplament, please check with me first. Also bear in mind that if I allow something and it turns out to be unbalancing, I reserve the right to adjust ("nerf") the feature in the future. I will, however, work with the player on this and try to come to some accomodationg that is acceptable to all.
